INSTALACION 

1. Se debe descargar e instalar XAMMP
2. Copiar el contenido de la carpeta en C:\xammp\htdocs
3. Subir servicios de APACHE y MySQL
4. Antes de Abrir configurar el archvo ".env" que esta en la raiz del proyecto
5. Crear base de datos y tablas desde el archivo pruebatecnica.sql
6. Instalar laravel con el composer : 
    composer global require laravel/installer
7. Crear un crud de Laravel con el comando :
    laravel new blog
9. Copiar ficheros a carpeta raiz    
8. Iniciar con las credenciales 
    Usuario : admin
    Password: admin
